# Plan 2026-01-21 — Gotify v1

**Date/Heure**: 2026-01-21
**Version**: v1

## Objectif

Envoyer une notification sur un serveur Gotify en cas de succès ou d'erreur lors de l'exécution du système.

## Étapes

1. **Créer un module de notification Gotify**
   - Créer un fichier dédié pour le client Gotify
   - Implémenter une fonction d'envoi de message avec les paramètres : URL serveur, token, titre, message, priorité

2. **Définir la configuration**
   - Ajouter les variables d'environnement nécessaires :
     - `GOTIFY_URL` : URL du serveur Gotify
     - `GOTIFY_TOKEN` : Token d'application Gotify
   - Rendre la fonctionnalité optionnelle (pas d'erreur si non configuré)

3. **Intégrer les notifications aux points de succès/erreur**
   - Identifier les points d'entrée/sortie du programme principal
   - Appeler la notification en cas de succès (priorité normale)
   - Appeler la notification en cas d'erreur (priorité haute)

4. **Ajouter les tests**
   - Test unitaire du client Gotify (avec mock HTTP)
   - Test de la désactivation quand non configuré

## Fichiers à toucher (scope)

- `src/notifications/gotify.py` (nouveau) — Client Gotify
- `src/main.py` ou point d'entrée principal — Intégration des appels
- `tests/test_gotify.py` (nouveau) — Tests unitaires
- `.env.example` — Documentation des variables d'environnement

## Risques

| Risque | Mitigation |
|--------|------------|
| Serveur Gotify indisponible | Timeout court + ne pas bloquer l'exécution principale |
| Token invalide | Log d'avertissement, pas d'exception fatale |
| Dépendance HTTP | Utiliser `requests` si déjà présent, sinon `httpx` ou stdlib `urllib` |

## Definition of Done

- [ ] Le module `gotify.py` existe et expose une fonction `send_notification()`
- [ ] Les variables `GOTIFY_URL` et `GOTIFY_TOKEN` sont documentées dans `.env.example`
- [ ] Une notification est envoyée en cas de succès
- [ ] Une notification est envoyée en cas d'erreur
- [ ] Les tests passent : `just ci`
- [ ] Pas de nouvelle dépendance sans justification
- [ ] La fonctionnalité est silencieusement désactivée si non configurée
